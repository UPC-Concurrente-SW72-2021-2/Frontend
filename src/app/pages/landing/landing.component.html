Obten informacion precisa acerca de los ultimos reportes de vacunados

Quienes somos

C19 Helper es una aplicacion concurrente y distribuida que utiliza el DataSet de la Plataforma Nacional de Datos Abiertos con el proposito de:
  • Optimización de consulta de personas vacunadas mediante filtros, ya sea por fecha de vacunación, centro de vacunación, tipo de vacuna, dosis o edad.
  • Identificar la información relevante del conjunto de datos utilizados, evaluarla y tratarla con enfoques éticos que cumplan con la finalidad de resolver el problema planteado.
  • Actualización eficaz de los datos debido a que el acceso a la información más reciente se refleja luego de varios días transcurridos.

    • Red Neuronal

    Este algoritmo puede clasificar los datos obtenidos del DataSet utilizando el paradigma de concurrencia. Ademas, presenta los mejores resultados tanto en tiempo de entrenamiento como en eficacia de prediccion.

    Como la entrenamos

    En el desarrollo de C19 – HELPER se han necesitado datos sobre pacientes vacunados para poder crear un motor de inteligencia artificial que pueda utilizarse para retribuir información detallada sobre un conjunto de parámetros establecidos.
    Para ello, se utilizó el conjunto de datos llamado “TB_VACUNACIÓN_COVID19”, el cual se obtuvo de la Plataforma Nacional de Datos Abiertos. Dicha plataforma permite encontrar y utilizar datos gubernamentales de una manera segura y confiable para el desarrollo de aplicaciones.
    Dado que el trabajo sigue los lineamientos de la metodología ágil, el conjunto de datos elegido se encuentra desplegado en el repositorio del proyecto, el cual podrá ser consumido directamente por la aplicación utilizando el enlace (en versión “raw”) que provee la plataforma de desarrollo colaborativo GitHub.
